# Angular CLI
## CLI

```npm i -g @angular/cli```

``` ng new myApp --routing --style=scss --skip-tests ```

``` cd myApp ```

``` ng config schematics.@schematics/angular.component.spec false ```

## Material Design

``` ng add @angular/material ```
